# Monefy Pro Clone

Web клон приложения Monefy Pro для управления личными финансами.

## Стек

- Next.js 15 + TypeScript
- Tailwind CSS
- Supabase (Auth + Database)
- Recharts
- Radix UI

## Установка

```bash
npm install
```

## Настройка

Создайте `.env.local` из `.env.local.example` и добавьте ключи Supabase.

## Запуск

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000)

## Деплой

Проект автоматически деплоится на GitHub Pages через Actions.
